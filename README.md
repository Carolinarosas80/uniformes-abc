# ABC Uniformes - Proyecto React + Vite + Sass (completo)

Generado según especificaciones de Carolina.
Incluye:
- Navbar hamburguesa responsive
- Rutas y páginas: /, /colegios, /empresas, /particulares
- Botón flotante de WhatsApp
- Animaciones modernas (CSS)
- SEO básico (meta description + titles por página)
- Footer institucional
- Tipografía: Poppins (Google Fonts)
- Estilos en un solo archivo Sass: src/styles/index.scss

## Instrucciones
1. `npm install`
2. `npm run dev`
3. Abrir `http://localhost:5173`

Reemplazar logo: `src/assets/logo-temp.png`
Actualizar WhatsApp: `src/components/WhatsAppButton.jsx` (número WA ya configurado)
